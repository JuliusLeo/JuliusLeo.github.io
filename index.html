Swap Variables

2017-10-22

Recently, a friend of mine mentioned the techniques that related to swapping two variables without additional memory. There are different approaches include mathamatical operation on two variables and bitwise operation on both the pointers and the variables. Since this question is pretty interesting, I’ll write it down here.

Problem Statement

Given two variables, swap them without additional memory such as temporary variables. Let x and y denote the two varaibles.

Approach 1: Mathamatical operation.

Given the two number, first we let the original values be x0 and y0. Then the following operation can swap the two variables.

x=x+y;
y=x-y;
x=x-y;

Proof:
‘’’
x=x0+y0;
y=x0+y0-y0=x0;
x=x0+y0-x0=y0;
‘’’
